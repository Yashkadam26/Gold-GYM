# Gold Gym - Premium Fitness Web Portal
A modern, multi-page gym website built with a focus on high-end UI/UX and smooth animations.

## ✨ Features
- **Dynamic Hero:** Auto-playing cinematic background with Anime.js text reveals.
- **Interactive Timeline:** A vertical "Legacy" scroll that tracks user progress from 1990 to 2026.
- **Custom Gallery:** Auto-sliding premium facility showcase.
- **Responsive Design:** Optimized for mobile and desktop fitness enthusiasts.

## 🛠️ Tech Stack
- HTML5 / CSS3 (Custom Properties & Flexbox)
- JavaScript (ES6+)
- [Anime.js](https://animejs.com/) for motion design
